适用说明

本组件使用原生JS构建，适用无法引入jquery库的情况。本组件依赖于固定的HTML结构，否则无法正常工作，具体HTML结构请参考下方“HTML结构”章节。

使用方法

HTML代码中标签页的ID为id="easyTab1"，对应内容区域应该是id="easyTab_1"。

标签区有两种样式，tabCurrent和tabNormal分别对应当前选中的标签与默认的标签。

HTML结构

<div class="easyTab">
  <span class="tabCurrent" id="easyTab1" onclick="setTab('easyTab',1,5)">
  <p>标签ONE</p>
  </span>
  <span class="tabNormal" id="easyTab2" onclick="setTab('easyTab',2,5)">
  <p>标签TWO</p>
  </span>
  <span class="tabNormal" id="easyTab3" onclick="setTab('easyTab',3,5)">
  <p>标签THREE</p>
  </span>
  <span class="tabNormal" id="easyTab4" onclick="setTab('easyTab',4,5)">
  <p>标签FOUR</p>
  </span>
  <span class="tabNormal" id="easyTab5" onclick="setTab('easyTab',5,5)">
  <p>使用iframe嵌入</p>
  </span>
</div>
  
<div class="tabsContent">
  <div class="item" id="easyTab_1"> 我排行老大 </div>
  <div class="item" id="easyTab_2" style="display:none;"> 二一添作五 </div>
  <div class="item" id="easyTab_3" style="display:none;"> 我是第三个标签页 </div>
  <div class="item" id="easyTab_4" style="display:none;"> 4朵金花 </div>
  <div class="item" id="easyTab_5" style="display:none;"><iframe src="http://www.baidu.com"></iframe></div>
</div>

Javascript代码

function setTab(name,cursel,n){
	for(i=1;i<=n;i++){
	var menu=document.getElementById(name+i);
	var con=document.getElementById(name+"_"+i);
	menu.className=i==cursel?"tabCurrent":"tabNormal";
	con.style.display=i==cursel?"block":"none";
	}
}

setTable(argument1,argument2,argument3)

参数1为标签与内容区ID的前缀，如下方HTML结构中的标签区的ID="easyTab1",内容区的ID="easyTab_1"，此处传入的字符串参数应该是easyTab

参数2为标签页的当前索引值，索引值从1开始。

参数3为标签页的数量值，如下方示例中含有5个标签，此处值应该是数值5。
