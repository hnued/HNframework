适用说明

本代码用于解决在页面中使用iframe嵌入页面时，页面内容很多出现滚动条问题，代码可以让iframe高度与其中的内容高度自适应。

兼容性问题

仅支持IE6+

使用方法

function setIframeHeight() {  
    var iObj = document.getElementById('ifarmeId'); //指定iframe的id
    iObj.height =  iObj.contentWindow.document.documentElement.scrollHeight; //设置高度与其内部容器的scrollheight一致
}
