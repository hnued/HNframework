适用说明

本组件适用于在一个页面展示多种类别的内容，内容区可以使用IFRAME嵌入另外的页面。本组件依赖于固定的HTML结构，否则无法正常工作，具体HTML结构请参考下方“HTML结构”章节。

使用方法

根据实际使用需要，指定样式为tabsArea的DIV宽高度，组件会依此完成内部相关元素的高宽度设置。

更改tabsArea的样式表文件为需要的宽高度或通过style指定标签页组件的宽高度。

<div class="tabsArea" style="width:100%;height:400px;">，通过前面这段代码，标签页整个区域的宽度是100%，高度是 400px。

第一步，粘贴HTML结构

<div class="tabsArea">
    <div class="tab-hd">
        <ul class="tab-nav" >
            <li>
                <a href="#tab1">精彩活动测试数据</a>
            </li>
            <li>
                <a href="#tab2">最热卖</a>
            </li>
            <li>
                <a href="#tab3">创意极品</a>
            </li>
            <li>
                <a href="#tab4">使用iframe嵌入内容</a>
            </li>
            <div style="clear:both"></div>
      </ul>
    </div>
    <div class="tab-bd">
        <div id="tab1" class="tab-content">
            第一个TAB内容
</div>
        <div id="tab2" class="tab-content">
            第二个TAB内容测试
</div>
        <div id="tab3" class="tab-content">
            第三个TAB内容
</div>
        <div id="tab4" class="tab-content">
           <iframe src="js_fixedLayer.html" frameborder="0" width="100%" height="100%"></iframe>//可以使用iframe嵌入页面
        </div>
    </div>
</div>

第二步，初始化组件

<script type="text/javascript">

  $(document).ready(function() {

$('ul.tab-nav').simpleTab();

});

</script>
